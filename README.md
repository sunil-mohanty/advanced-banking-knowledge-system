Overview

This system implements cutting-edge retrieval techniques to answer complex banking questions:

- GraphRAG: Relationship-based retrieval using knowledge graphs
- Vector Search: Semantic similarity using embeddings
- RAFT: Relevance evaluation for document filtering
- Intelligent Fusion: Adaptive weighting of retrieval methods
- GPT-4 Synthesis: Comprehensive answer generation
